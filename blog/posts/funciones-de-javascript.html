Funciones sobre los array

Agrega un elemento en el primer lugar de un array → Retorna la cantidad de elementos 

["Pedro", "Joaquin"].unshift("Damian")
//Te queda ["Damian", "Pedro", "Joaquin"]

Agrega un elemento en el ultimo lugar del array → Retorna la cantidad de elementos 

["Pedro", "Joaquin"].push("Damian")
//Agrega "Damian" y queda ["Pedro", "Joaquin", "Damian"]

Elimina el ultimo elemento de un array y lo retorna 

[1,2,3,4].pop()
//Devuelve 4 y queda [1,2,3]

Elimina el primer elemento de un array y lo retorna 

[1,2,3,4].shift()
//Devuelve 1 y queda [2,3,4]
