Generar la informacion de forma programatica

Creamos un archivo index.js y dentro colocam 

module.exports = () => {
  const data = { perros: [] };
  // Create 1000 perros
  for (let i = 0; i < 1000; i++) {
    data.perros.push({ id: i, name: `user${i}` });
  }
  return data;
};

Luego, lo ejecutamos con: npx json-server index.js
